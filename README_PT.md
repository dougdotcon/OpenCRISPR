# OpenCrisprSystem

![Header do OpenCrisprSystem](imgs/header.png)

O **OpenCrisprSystem** é uma coleção de sistemas de edição de genes gratuitos, abertos e projetados por IA, desenvolvidos pela Profluent Bio. Ele serve como um recurso abrangente para pesquisadores que buscam nucleaseos programáveis e avançados adaptados para tarefas de edição específicas.

## Visão Geral do Sistema

A versão principal, **OpenCRISPR-1**, representa uma inovação no design de proteínas impulsionado por IA. Ela consiste em uma proteína semelhante à Cas9 e um RNA guia compatível, ambos gerados usando grandes modelos de linguagem (LLMs). Diferente das variantes naturais de Cas9, o OpenCRISPR-1 apresenta centenas de mutações únicas, mantendo a arquitetura prototípica de uma nuclease do Tipo II.

**Características Principais:**
*   **Projetado por IA:** Gerado inteiramente pelas LLMs da Profluent.
*   **Alta Compatibilidade:** Funciona como uma substituição direta para protocolos SpCas9 com preferência por PAM NGG.
*   **Formatos Versáteis:** Pode ser adaptado para formatos desativados ou de *nickase* para edição de base, edição de primo ou edição epigenética.

## Licenciamento e Uso

O OpenCrisprSystem está comprometido em democratizar o acesso às tecnologias de edição de genes.

*   **Uso Comercial:** O OpenCRISPR-1 está disponível gratuitamente para uso comercial, desde que os usuários aceitem o contrato de licença.
*   **Diretrizes Éticas:** A licença exige o uso ético das ferramentas e que os usuários reconheçam a Profluent Bio como desenvolvedora. Também é necessário notificar a empresa se produtos derivados do OpenCRISPR avançarem para etapas clínicas ou comerciais.
*   **Propriedade Intelectual:** Embora a sequência esteja publicamente disponível através do *pre-print*, a propriedade intelectual foi registrada.
